Sudoku Board Generator

This java code generates a solved sudoku board.
To run the program type "java Sudoku.java" in the terminal and press enter.
In the main method, the board is created using the solve method. The solve method uses the isFilled method, which goes through the board filling it with valid numbers,to fill the board. The isValid method is used to check that there aren't repeating numbers in a row, column, or 3x3 box.
